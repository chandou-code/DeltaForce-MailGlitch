# 三角洲半自动卡邮箱

无需删除地图重新下，半自动卡邮箱工具。省去大量手动操作步骤，只需修改配置文件并跟随脚本提示操作即可完成卡邮箱，支持循环卡。

## 脚本提示

自动化过程中，脚本会在关键节点给出提示，按提示操作即可：

| 阶段 | 提示内容 |
|------|----------|
| 阶段1 - 游戏启动后 | 请裸装进入长弓然后在选人界面大退 |
| 阶段3 - 游戏启动后 | 把要卡的东西放在身上然后大退 |
| 阶段6 - 游戏启动后 | 上号重连进长弓自雷以后大退，完成后自动循环回到第1阶段 |

## 自动化流程

脚本会自动检测 DeltaForce 进程的运行状态，在你每次大退游戏后自动执行对应操作，全程循环执行：

```
阶段1: 等待第一次大退
阶段2: 执行脚本修改（重命名 pak 文件）
阶段3: 等待第二次大退
阶段4: 执行脚本恢复（还原 pak 文件 + 替换 dat 文件）
阶段5: 重启 WeGame（先关闭再启动）
阶段6: 等待第三次大退
阶段7: 重启 WeGame
↓ 完成后自动回到阶段1，开始下一轮
```

## 修改配置

打开 `config.json`，根据你的实际情况修改以下配置：

| 配置项 | 说明 |
|--------|------|
| `PAK_DIR` | 游戏 pak 文件所在目录 |
| `RAIL_DIR` | rail_files 目录路径 |
| `WEGAME_PATH` | WeGame 启动程序路径 |
| `PAK_ORIG` | 原始 pak 文件名 |
| `PAK_NEW` | 修改后的 pak 文件名 |
| `DAT_NAME` | dat 文件名（默认 package_info.dat） |
| `DAT_SRC_DIR` | 用于替换的 dat 源文件所在目录 |
| `WEGAME_PROCESSES` | 需要关闭的 WeGame 进程列表 |

> `DAT_SRC_DIR` 目录下需要放一份准备好的 `package_info.dat` 文件，恢复时会用它覆盖游戏目录里的原文件。

## 启动脚本

```bash
python pak_manager.py
```

选择功能：

```
===== 功能菜单 =====
1. 修改        # 手动执行脚本修改（重命名 pak 文件）
2. 恢复        # 手动执行脚本恢复（还原 pak 文件 + 替换 dat 文件）
3. 自动化      # 全自动循环卡邮箱
0. 退出
====================
```

输入 `3` 进入自动化模式，之后跟随提示操作即可。

## 其他说明

- 按 `Ctrl+C` 可随时停止自动化模式
- 无需安装额外 Python 依赖，仅使用标准库
- 脚本每秒检测一次游戏进程状态
- `close_wegame.py` 可单独使用来关闭/启动 WeGame：
  - `python close_wegame.py` 关闭 WeGame
  - `python close_wegame.py start` 启动 WeGame
